Инструкция:
сперва нужно пройти мини игру с словами, попасть к нему модно нажав на табличку с огнем
Управление: Нужно нажимать ЛКМ сперва на слово потом на перевод по порядку, после того как все сделано нажать ЛКМ далее в нижнем правом углу

Потом можно перейти на уровень с демонами нажав на открывшийся деревню.
Управление: wsda перемещение, нажав ЛКМ, есть рывок на пробел space

Задумка: Выйграть уровень собирая слова в правильном порядке для построение слов.
