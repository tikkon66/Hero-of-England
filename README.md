Игра (обучающий 2D проект)
Описание проекта:
Игра была разработана совместно с одногруппником в учебных целях и из интереса к геймдеву. Проект не завершён до конца из-за ограничений по времени и ресурсу команды.

!Важно: мобильная адаптация отсутствует.

Инструкция по запуску и прохождению:

1. Мини-игра со словами:
Первый этап игры — обучающая мини-игра со словами.
Для входа в мини-игру необходимо нажать на табличку с огнём( Сверху в середине)
Управление:
ЛКМ (левая кнопка мыши) — выбор слова
Сначала нажимается слово, затем его перевод по порядку
После выполнения задания необходимо нажать ЛКМ в нижнем правом углу для продолжения

2. Переход к основному уровню
После завершения мини-игры открывается доступ к деревне.
Переход осуществляется через открывшуюся локацию (деревня)

4. Уровень с демонами
Основной игровой уровень с элементами экшена.
Управление:
W, S, A, D — перемещение персонажа
ЛКМ — атака / действие
Space — рывок (dash)
Игровая механика и задумка

Основная идея игры — обучение через геймплей.
Игрок должен(не реализована):
собирать слова в правильном порядке
формировать правильные комбинации для прохождения уровня
выполнять задания для получения наград
В конце уровня игрок получает кристаллы, которые можно использовать для получения скинов.
